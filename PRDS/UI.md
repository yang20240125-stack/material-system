# 🎨 企业物资管理系统 UI 设计文档

## 1. 推荐前端技术栈与组件库
请基于以下技术栈进行开发，确保代码透明且对 AI 生成友好：

* **核心框架**：Next.js (App Router) + React + TypeScript。
* **CSS 框架**：Tailwind CSS。
* **UI 组件库**：**shadcn/ui** (直接将源码集成到项目中，便于定制)。
* **图标库**：Lucide React。

## 2. 全局设计规范
* **设计风格**：现代企业级、极简主义 (Minimalist)、高信息密度。
* **色彩系统 (Color Palette)**：
    * 主色调 (Primary)：品牌蓝 (与钉钉生态融合，如 `bg-blue-600`)，用于主按钮和 AI 高亮。
    * 背景色 (Background)：极浅灰 (`bg-slate-50` 或 `#F9FAFB`)，突出内容卡片。
    * 文字色 (Text)：主标题纯黑 (`text-slate-900`)，次要信息灰 (`text-slate-500`)。
    * 状态色 (Status)：危险/删除用红色，警告/库存不足用橙色，成功/充足用绿色。
* **排版 (Typography)**：系统默认无衬线字体 (Inter 或 系统原生字体)，卡片标题加粗 (`font-semibold`)。
* **交互原则**：所有涉及到数据库修改的操作，必须有二次确认弹窗或在 AI 对话流中生成确认卡片。

## 3. 核心布局结构 (Layout)
采用经典的“顶侧导航 + 主内容区”布局，兼顾 PC 端信息展示和响应式收缩。

1.  **顶部导航栏 (Top Navbar)**：Logo、全局搜索、当前登录用户信息（头像/姓名）、钉钉免登状态标识。
2.  **左侧菜单栏 (Sidebar)**：物资大厅（所有人可见）、流转台账（Admin可见）、系统设置（Admin可见）。
3.  **主工作区 (Main Content)**：展示具体的列表、卡片或表单。
4.  **悬浮组件 (Floating Action)**：右下角常驻的 AI 助手入口。
